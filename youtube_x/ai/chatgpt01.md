    - そして、YouTubeなどの自己啓発動画、パラノイア、内田、THINKなどを
    - 参考にして、できた習慣が、
    - 早朝4時に起きて、ベッドメイキング、水1杯を飲む、
    - 1.筋トレ(腕立て100回, 腹筋250回, スクワット100回, 体幹2分20秒)
    - これを科学的にや頻度、食事法をこれから取り入れつつも、約10分でやる
    - 2.散歩・ランをおこなう、これは10~15分程度
    - 3. 冷水シャワーを浴びて、
    - 4.スキンケアとして、化粧水などを塗る
    - 5. ストレッチをする(体、美容、目、鼻、顔などを含めて)

    - この1~8のルーティンを朝・昼・夜の3回行う。
    - ただし、最小単位も考えているので、すべて最小単位だった場合、
    - 1分以内に1~8のルーティンすべてを終えることができる。
    - Deep Work を始める。ここでは90分(作業) + 10分(瞑想)をおこない、
    - これは「カエルをたべてしまえ」「思考の整理学」の朝飯前、
    - 「Deep Work」を参考にしたものだ。なので、
    - 1~8のルーティンを先にやるか、Deep Workを先にやるかは要相談。
    - ここでは、1日の中で一番難しい作業をおこなう。
    - それを終えると、まだ朝なので、勉強、資格、などをする。
    - ここでは、プログラミングの勉強(TIL(GitHub))や、
    - 資格の勉強(Ankiを使用)などを活用する。
    - これらを終えて、朝8時くらいになると、洗濯、風呂掃除、部屋掃除、
    - 8.はみがき、を一気に行う、10-15分ほどで終わる。
    - そのあとは、ポモドーロ・テクニックを使用して、
    - 25分(仕事) + 5分(瞑想+休憩)をおこなう。
    - このとき、トイレ休憩以外何もなければ、目を瞑って瞑想をおこなう。
    - これをひたすら繰り返し、午後12時の鐘が鳴ると
    - また、1~8のルーティンをこなす。ここでも最小単位も含める(1分で終わる)
    - そして、昼食を食べるのだが、筋トレ、美容スキンケア、薄毛対策、栄養学、
    - 節約(出費を抑える)、などを考慮に入れた、つくりおきや、
    - YouTubeでの1日100円弁当を参考にして、1回の料理で5日分の料理をつくっておき、
    - それをziplock や タッパーにいれて、冷凍庫にしまう。
    - これを取り出して、チンをして、昼食を食べる。
    - 皿洗いをした後に、30分後歯磨きをして、また、
    - 25分(仕事) + 5分(瞑想 + 休憩)のポモドーロ・テクニックを活用して、
    - 午後17時まで仕事をして、1.筋トレをおこない、お風呂をためて
    - 2.散歩・ランをおこない、図書館に行き(このために図書館が近くにある
    - 家に住むことになる)、日経新聞を読み,1s1p読書を簡単に行う。

    - 抑えるはたらきになる。(「Atomic Habis」に倣って)

    - そして、この間に7.日記、読書、勉強をする。
