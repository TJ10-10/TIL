07: CSS Variables & Theme

変数で色を一元管理

:root に宣言した --bg, --text, --primary などを どこでも再利用できます。テーマはクラスで上書きすれば即切り替え可能。

primary

ダークテーマの適用

.theme-dark を body に付けるだけで、変数が切り替わり全体の色・影・境界が変化します。 アニメーションの知識と組み合わせると、滑らかなトランジションにできます。

保守性と拡張性

色や影を変えたいとき、1箇所の変数を変えるだけで全体に反映。コンポーネント設計と相性抜群です。

変数のスコープは継承されます。特定のセクションだけ色味を変えたいなら、親要素に変数を再定義して子に伝播させられます。
例: .section { --primary: #ff6b6b; }
