# 2025-11-05: Go Hello WorldとGitHubへのpush

## 学んだこと
- `go run hello.go` で実行できる
- `git remote add origin` は初回に必要
- `git pull --allow-unrelated-histories` で履歴を統合

## コマンド例
```bash
git remote add origin git@github.com:TJ10-10/TIL.git
git pull origin master --allow-unrelated-histories --no-rebase
git push -u origin master
